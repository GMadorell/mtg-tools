Card Importer Tool

Enter a set code, followed by the collector number – e.g. kld247 or khc082.
For cards without set code and card number, switch to “Name + Year”.
Toggle foil/proxy/language if needed, set quantity, then press “Enter” or click “Add”.
You can edit quantity/foil/proxy/language in the table. If quantity goes to 0, that row is removed.
If two rows become the same after editing (same set code, collector #, foil, proxy, language), they merge.
Prices come from Scryfall in EUR.
